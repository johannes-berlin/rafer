Webflow Integration - HTML Struktur Referenz

Hinweis: Diese Datei dient nur als Referenz für die HTML-Struktur, die Sie in Webflow erstellen müssen.

1. Hero Section (Scatter Animation)

Container: Section Element mit CSS Class hero_main_wrap
Nest-Bereich: Div Block mit CSS Class nest-area
Text: Heading Element mit Attribut data-adhd="true" 
<section class="hero_main_wrap">
  <div class="nest-area"></div>
  <h1 data-adhd="true">Ihr Text hier</h1>
</section>

2. Challenges Section (Card Animation)

Hauptcontainer: Section Element mit CSS Class challenges_wrap
Cards-Container: Div Block mit CSS Class challenges_cards_wrap
Einzelne Cards: Div Block mit CSS Class challenge_card
Card-Inner: Div Block mit CSS Class challenge_card_inner 
<section class="challenges_wrap">
  <div class="challenges_cards_wrap">
    <div class="challenge_card">
      <div class="challenge_card_inner">
        <h3>Card Titel</h3>
        <p>Card Inhalt</p>
      </div>
    </div>
    <div class="challenge_card">
      <div class="challenge_card_inner">
        <h3>Card Titel</h3>
        <p>Card Inhalt</p>
      </div>
    </div>
    <!-- Weitere Cards... -->
  </div>
</section>

3. Modal Section (Clip Animation)

Modal-Wrapper: Div Block mit CSS Class modal-wrap
Modal-Hintergrund: Div Block mit CSS Class modal-bg
Sidebar: Div Block mit CSS Class sidebar
Modal-Inhalte: Div Block mit Attribut data-modal="founders" oder data-modal="about"
Trigger-Buttons: Button Element mit Attribut data-modal-cta="founders" oder data-modal-cta="about"
Schließen-Buttons: Button Element mit Attribut data-modal-close 
<div class="modal-wrap">
  <div class="modal-bg"></div>
  <div class="sidebar">
    <div data-modal="founders">
      <h2>Gründer</h2>
      <p>Inhalt...</p>
      <button data-modal-close>Schließen</button>
    </div>
    <div data-modal="about">
      <h2>Über uns</h2>
      <p>Inhalt...</p>
      <button data-modal-close>Schließen</button>
    </div>
  </div>
</div>

<!-- Trigger Buttons -->
<button data-modal-cta="founders">Gründer anzeigen</button>
<button data-modal-cta="about">Über uns</button>

4. CSS Classes Übersicht

Wichtig: Alle diese CSS Classes müssen in Webflow erstellt werden

Hero Section:

  • hero_main_wrap - Hauptcontainer
  • nest-area - Nest-Bereich für Animation

Challenges Section:

  • challenges_wrap - Hauptcontainer
  • challenges_cards_wrap - Cards-Container
  • challenge_card - Einzelne Card
  • challenge_card_inner - Card-Inhalt

Modal Section:

  • modal-wrap - Modal-Container
  • modal-bg - Modal-Hintergrund
  • sidebar - Modal-Sidebar

5. Data Attributes Übersicht

Wichtig: Diese Attribute müssen in Webflow hinzugefügt werden

Scatter Animation:

  • data-adhd="true" - Text für Scatter-Animation

Modal Animation:

  • data-modal="founders" - Modal-Inhalt für Gründer
  • data-modal="about" - Modal-Inhalt für Über uns
  • data-modal-cta="founders" - Button für Gründer-Modal
  • data-modal-cta="about" - Button für Über uns-Modal
  • data-modal-close - Schließen-Button

6. Webflow Integration Schritte

  1. GSAP Scripts hinzufügen: Site Settings → Custom Code → Head Code
  2. Animationen einbetten: Site Settings → Custom Code → Footer Code
  3. HTML-Struktur erstellen: Alle Elemente mit korrekten CSS Classes
  4. Data Attributes hinzufügen: Alle notwendigen data-* Attribute
  5. CSS-Styling: Alle Styles in Webflow Designer
  6. Testing: Preview und Live-Testing
